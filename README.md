binary-static
=============

This repository contains the static Javascript and CSS content of the Binary.com website.


CSS & JS Minification
====================
To automate the stuff for css and javascript minification

**Requirements**

- Ruby, RubyGems
- Node.js and NPM

**Installation**

- Go to project root
- `gem install compass`
- `npm install`
- `npm install -g grunt-cli`
- `grunt`
